#  chopin-cli
![Npm 版本](https://img.shields.io/badge/chopin-cli_v0.0.1-green)

## 📖简介
- 用于快速搭建前端项目的命令行工具

## 📦 安装

```bash
npm install chopin-cli -g
```
## 🚩使用

```bash
# 使用模板创建新项目
chopin create 

# 执行项目文件夹名称并选择模板创建新项目
chopin create [projectName]

# 更新脚手架到最新版本
chopin update

# 查看脚手架版本
chopin -v
chopin --version
```
